## Dependencies

### Production

- PostgreSQL v16.3
- Tern - (DB migrations) https://github.com/jackc/tern


## Running

Then the app can be run:

```
cd ./webapp
go run .
```


## Info

Times stored in the database should be assumed to be UTC.
